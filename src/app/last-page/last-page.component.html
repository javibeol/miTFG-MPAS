Si quieres volver atrás, pulsa el siguiente botón:

(Esto sobreescribirá los archivos "namelist.init_atmosphere" y "streams.init_atmosphere" para

poder usarlos en el apartado anterior)



Contínua con los siguientes pasos:

1] Descarga el archivo "namelist.init_atmosphere" ya configurado:

wget http://localhost:3000/files/namelist.init_atmosphere

2] Descarga el archivo "streams.init_atmosphere" ya configurado:

wget http://localhost:3000/files/streams.init_atmosphere

3] Una vez descargado los archivos ejecuta el siguiente comando:

./init_atmosphere_model

4] Descarga el archivo namelist.atmosphere ya configurado:

wget http://localhost:3000/files/namelist.atmosphere

5] Descarga el archivo streams.init_atmosphere ya configurado:

wget http://localhost:3000/files/streams.atmosphere

6] Una vez descargados los archivos, ejecuta el siguiente comando:

./atmosphere_model

7] Para ejecutar en paralelo:

Asegúrate de comprobar en cuántas partes permite estar dividida la malla que has descargado y descomprimido.

Para verlo, puedes ejecutar este comando:

ls -l {{dirCasos}}/{{dirCaso}}/{{nombre}}


(El NPROC es el número de procesos MPI que quieras crear dependiendo de las partes

en las que la malla que has elegido permita dividirse)

Ejecuta el siguiente comando:

mpiexec -n NPROC ./atmosphere_model
