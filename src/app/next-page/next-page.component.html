Si los datos sí son los deseados, realiza los siguientes pasos:

1] Sitúate en el directorio del caso dónde quieres realizar la ejecución:

cd {{dirCaso}}

2] Ejecuta el siguiente comando para descargar el archivo de la malla:

wget https://www2.mmm.ucar.edu/projects/mpas/atmosphere_meshes/{{nombre}}.tar.gz

3] Ejecuta el siguiente comando para descomprimir el archivo que acabas de descargar:

tar -xzf {{nombre}}.tar.gz

4] Haz clic en el enlace para descargar el archivo namelist.input ya configurado con los datos seleccionados:

Descargar namelist.input

5] Ejecuta el siguiente comando para ejecutar la utilidad de rotación de la malla con el fichero de configuración namelist.input que acabas de descargar:

grid_rotate {{nombre}}.grid.nc {{nombre}}.grid_out.nc
